Вход
На входе регулярно грузим и обрабатываем массивы знаний о различных структурах маркетов:
  • Структуры маркетов
  • Категории товаров в маркетах
  • Данные из апи по аттрибутам категорий
  • Спарсенные данные
  • Накопленные данные о связях категорий между маркетами
Движок и рендер
Движок методами разного уровня доверия связывает между собой категории, атрибуты, значения из всех маркетов

Через структуры импорта рендерим товары в каталог, через структуры экспорта подготавливаем к обратной выгрузке в маркеты.
Выход
Полученные карточки можно редактировать и сохранять сразу во всех маркетах. Можно создавать карточки в непредставленных маркетах. Удачный экспорт с хорошим рейтингом сохраняется в облако.
